# Colombia Sin Venda — Día 14: La Deuda que Nos Dejaron

PWA educativa sobre el crecimiento de la deuda pública colombiana 2022–2026.

## Datos clave
- Deuda total: $804,9B (ago-2022) → $1.238,2B (feb-2026) = +54%
- Tasa máxima pagada: 14,79% en TES largo plazo (mayo 2026)
- Crecimiento promedio: $319.000 millones/día · $222 millones/minuto
- Deuda de corto plazo 2025: +368% vs 2024
- Servicio deuda 2026: $130 billones = $1 de cada $3 del recaudo

## Fuentes
MinHacienda, El Colombiano, Portafolio, Infobae Colombia, La República, Contraloría General, Banco de la República.
